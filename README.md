# Associate Data Scientist (Python) - Sertifikasi Nasional 

Halo! Selamat datang di repositori dokumentasi belajar saya. 

Repositori ini saya buat khusus untuk menyimpan seluruh materi, tugas, dan proyek yang saya kerjakan selama mengikuti program pelatihan **Associate Data Scientist (Python)** tingkat Nasional (KKNI Basic). Program ini sangat berkesan bagi saya karena memberikan wawasan mendalam tentang bagaimana data bisa diolah menjadi solusi AI yang nyata.

##  Ringkasan Program
- **Kategori:** Associate Data Scientist
- **Level Sertifikasi:** KKNI Basic

##  Apa yang Saya Pelajari?
Pelatihan ini mencakup banyak hal, namun ada dua sesi utama (*Live Sessions*) yang menjadi inti dari repositori ini:

1. **Strategi Mengoptimalkan Model AI**
   Sesi ini membahas bagaimana cara kita melakukan tuning dan pemilihan strategi yang tepat agar model AI yang dihasilkan tidak hanya akurat, tapi juga efisien untuk kebutuhan masa depan.

2. **End-to-End Data Science Solution**
   Di sesi ini, saya belajar alur kerja lengkap seorang Data Scientist. Mulai dari memahami masalah, mengolah data mentah, hingga mengimplementasikan model AI secara menyeluruh (*End-to-End*).

##  Tools & Teknologi
- **Bahasa Pemrograman:** Python
- **Lingkungan Kerja:** Live Coding Python, Jupyter Notebook
- **Fokus Utama:** AI Model Optimization, Machine Learning Workflow
